Module-6: Ansible Assignment – 3

Tasks To Be Performed:
1. Create 2 Ansible roles
2. Install Apache2 on slave1 using one role and NGINX on slave2 using the other role
3. Above should be implemented using different Ansible roles

1.     Create 2 Ansible roles

 

To create ansible roles:

Go inside the directory  cd /etc/ansible/roles

sudo ansible-galaxy init apache2

sudo ansible-galaxy init nginx

 

2. Install Apache2 on slave1 using one role and NGINX on slave2 using the other role

path of Ansible Apache role

cd /etc/ansible/roles/apache2/tasks/

path of Ansible nginx role

cd /etc/ansible/roles/nginx/tasks/

add the yml playbook files inside tasks

---

- name: apache2
apt: name=apache2 state=latest

sudo nano main.yml

---
- include_tasks:install.yml

 

---

- name: nginx
apt: name=nginxstate=latest

 

sudo nano main.yml

---
- include_tasks:install.yml

finally create a playbook to execute the tasks of the role

sudo nano assign3.yml

 

---

- name: installing apache2

  hosts: test

  become: true

  roles:

  - apache

- name: installing nginx

  hosts: prod

  become: true

  roles:

  - nginx

 

ansible-playbook assign3.yml (to execute the ansible playbook)
